# NextTrip
Sistema de informações e curiosidades sobre países

## Alguns campos que podemos preencher com dados dos Países: 

* Nome
* População
* Curiosidade
* Imagem da Bandeira
* Status - Já Visitou, Quer Visitar e se Não Quer Visitar


## Tecnologias utilizadas

* Django – framework web principal
* Django REST Framework – para criação de APIs
* Pillow – para manipulação de imagens (bandeiras)
